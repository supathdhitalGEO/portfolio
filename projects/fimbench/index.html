FIMbench

Project BriefUpdated May 8, 2026

FIMbench organizes benchmark flood-inundation data and related tooling for comparing model outputs at scale. It helps make evaluation datasets easier to browse, document, and reuse.

BenchmarkingAWS S3StreamlitFIM
ShareLinkedInX

What It Does

  • Organizes multi-tier benchmark datasets for flood-inundation comparison, hosted on AWS S3 for accessible, scalable storage.
  • Connects database structure with a companion Streamlit web app for interactive dataset exploration.
  • Supports repeatable evaluation and data discovery across FIM research workflows.

Why It Matters

Benchmarking only works when reference data is findable and structured. FIMbench helps connect large datasets with clear evaluation workflows, bridging the gap between raw reference products and the standardized comparisons needed for rigorous, reproducible FIM research.
